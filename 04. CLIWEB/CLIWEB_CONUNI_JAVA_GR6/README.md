# CLIWEB_CONUNI_JAVA_GR6

Front-end web client for the CONUNI SOAP Conversor project. Built with React + Vite.

This repository contains the web UI used to consume the `ConversorWS` SOAP service.

Quick start:

```bash
cd "04. CLIWEB/CLIMOV_CONUNI_JACA_GR6"
npm install
npm run dev
```
